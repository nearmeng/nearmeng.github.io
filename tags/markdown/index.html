markdown语法简介

已经好久没有开始写博客了，自从上次的博客域名expire之后，就没有了写博客的冲动。最近，突然又想拾起笔来，重新做人:) 于是就开始找寻可以记录信息的地方，大大小小的博客网站虽然多，但是能够还自己一份安宁的地方却是少之又少，就在偶然的查找资料的过程中，我发现了 scriptogr.am 这个网站，可以说真是缘分，它使用的是我一直使用的 Dropbox 服务，只需要在Dropbox中存放你自己编写的md格式的文档，经过scriptogr同步之后，就可以将md格式转换为html输出，变成你博客的一篇文章。

网站的界面清爽简洁，也真是我一直希望的博客的格式，节约却不简单，哈哈。在熟悉了博客的环境之后，我去熟悉了以下传说中的简单的MarkDown文本编辑语言，这是一种非常易读易写的文本编辑语言，有关于它的介绍可以这里查看->,它的学习和使用也是非常的简单和人性化，使用教程可以在这里查看->,借用网上的一句话说，那就是，这是一种你可以完全掌握的语言。学习Markdown语言，主要需要记住的几个语法分别是：

文章换行

敲两个回车(或者是先敲两个以上的空格再加个回车)

文章标题

#   一级标题
##  二级标题
### 三级标题 等

文本标注

*A* A倾斜
**A** A加重
